# 贪吃蛇游戏

一个贪吃蛇小游戏，目前未做移动端适配。

游戏规则：

1. 通过方向键控制贪吃蛇的移动方向，吃到苹果，增加得分；
2. 如果头部碰到墙壁或自身，游戏结束。

实现功能：

1. 方向键控制贪吃蛇的移动方向；
2. 吃到苹果后，分数刷新，苹果位置刷新；
3. 蛇头碰撞检测，碰到墙壁或自身后游戏结束；
4. 苹果刷新位置检测，保证苹果刷新位置不会与蛇身重合。
